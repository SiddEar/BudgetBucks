# BudgetBucks

A personal finance tracker I'm building from scratch to learn 
Python, SQL, and data analysis.

## What it does
- Tracks income and expenses from a CSV dataset
- Analyzes spending by category and month
- (Coming soon) Visualizes trends with charts

## Why I'm building this
I want to understand where my money goes and learn real data skills in the process

## Tech Stack
- Python + Pandas(data loading and analysis)
- SQL/SQLite (database)
- Github(Version Control)

## Project Status
Week 1 - Setting up data pipeline and loading first data set

## Folder structure
data/ -> transaction CSV Files
scripts/ -> Python analysis scripts
database/ -> SQL database files
